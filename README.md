jQuery QueryBuilder
===================

jQuery plugin offering an interface to create complex queries.

**Still in developpement**

### Requirements
 * Bootstrap 3.x CSS
 * jQuery >= 1.9
 * (optional) MomentJS
 * (optional) any widgets library like jQuery UI

### Browser support
 * Internet Explorer >= 10
 * Mozilla FireFox ??
 * Google Chrome ??
 * Opera ??
 * Safari ??
